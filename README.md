# AI Caption Generator

## 📌 Description
AI Caption Generator is a deep learning-based application that generates descriptive captions for images. The system uses a Convolutional Neural Network (CNN) to extract features from images and a Long Short-Term Memory (LSTM) network to generate natural language captions.

## 🚀 Features
- Generates captions for input images  
- Uses CNN for image feature extraction  
- Implements LSTM for sequence generation  
- Combines computer vision and NLP techniques  

## 🛠️ Technologies Used
- Python  
- TensorFlow / Keras  
- CNN  
- LSTM  
- Natural Language Processing (NLP)
